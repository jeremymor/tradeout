# Sound Files

This directory should contain the following retro Mac-style sound effects:

- `key-click.mp3` - Subtle click sound for typing
- `submit.mp3` - Button press sound
- `wrong-beep.mp3` - Error/incorrect guess sound
- `win-chime.mp3` - Success/celebration sound

## Where to Find Sounds

You can find free retro/8-bit sound effects at:
- https://freesound.org/
- https://pixabay.com/sound-effects/
- https://mixkit.co/free-sound-effects/

Search for terms like:
- "retro click"
- "8-bit beep"
- "chiptune success"
- "pixel sound"

Make sure to use royalty-free sounds or create your own!
